# Accounting-Ledger-Demo
Demo for Accounting Ledger

The Accounting Ledger is able to read existing files in .csv format, or write new files to csv format.  Users are able to pick a ledger size up to 20 and as each entry is made, with a date, description and amount, the ledger keeps a running balance for the ledger.  On top of this, the ledger is sorted by date of each entry, and has a feature to view either debits or credits only.
